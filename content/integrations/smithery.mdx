---
title: Smithery
description: Install Docfork MCP via Smithery
---

To install Docfork MCP Server for any client automatically via [Smithery](https://smithery.ai/server/docfork):

```bash  theme={null}
npx -y @smithery/cli@latest install docfork --client <CLIENT_NAME> --key <YOUR_SMITHERY_KEY>
```

You can find your Smithery key in the [Smithery.ai webpage](https://smithery.ai/server/docfork).
