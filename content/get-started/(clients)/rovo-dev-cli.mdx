---
title: Rovo Dev CLI
description: Install Docfork MCP in Rovo Dev CLI
---

Edit your Rovo Dev CLI MCP config:

```bash  theme={null}
acli rovodev mcp
```

#### Remote Server Connection

```json  theme={null}
{
  "mcpServers": {
    "docfork": {
      "url": "https://mcp.docfork.com/mcp"
    }
  }
}
```

#### Local Server Connection

```json  theme={null}
{
  "mcpServers": {
    "docfork": {
      "command": "npx",
      "args": ["-y", "docfork", "--api-key", "YOUR_API_KEY"]
    }
  }
}
```
