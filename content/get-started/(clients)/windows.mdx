---
title: Windows
description: Install Docfork MCP on Windows
---

The configuration on Windows is slightly different. Use `cmd` to run npx:

```json  theme={null}
{
  "mcpServers": {
    "docfork": {
      "command": "cmd",
      "args": ["/c", "npx", "-y", "docfork", "--api-key", "YOUR_API_KEY"],
      "disabled": false,
      "autoApprove": []
    }
  }
}
```
