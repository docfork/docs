---
title: Using Docker
description: Run Docfork MCP with Docker
---

1. Create a `Dockerfile`:

```dockerfile  theme={null}
FROM node:18-alpine
WORKDIR /app
RUN npm install -g docfork
CMD ["docfork"]
```

2. Build the image:

```bash  theme={null}
docker build -t docfork .
```

3. Configure your MCP client:

```json  theme={null}
{
  "mcpServers": {
    "docfork": {
      "command": "docker",
      "args": ["run", "-i", "--rm", "docfork-mcp"],
      "transportType": "stdio"
    }
  }
}
```
