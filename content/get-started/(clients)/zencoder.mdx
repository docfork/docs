---
title: Zencoder
description: Install Docfork MCP in Zencoder
---

1. Go to the Zencoder menu (...)
2. Select Agent tools
3. Click on Add custom MCP
4. Add the name and configuration:

```json  theme={null}
{
  "command": "npx",
  "args": ["-y", "docfork", "--api-key", "YOUR_API_KEY"]
}
```

5. Click Install.
